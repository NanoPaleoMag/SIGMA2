# SIGMA2

Updated version of Spectral Interpretation using Gaussian Mixtures and Autoencoder (SIGMA)

## Installation


1. Create a [**Python>=3.10**](https://www.python.org/) environment with [**conda**](https://docs.conda.io/en/latest/):
```bash
conda create -n sigma2 python=3.10
conda activate sigma2
```

2. Install the requirements from the `requirements.txt` file using [**pip**](https://pypi.org/project/pip/):
```bash
pip install -r requirements.txt
```


## Getting Started

1. Start a jupyter lab seesion to start running **SIGMA**. In your `conda` terminal, run:
```bash
jupyter lab
```

2. Navigate to the `tutorials` folder in the `jupyter` session, and open one of the tutorials notebooks. It is recommended to start with `Interactive_SEM_tutorial.ipynb`
   
3. Follow the instructions in the notebook to learn a typical **SIGMA** workflow
